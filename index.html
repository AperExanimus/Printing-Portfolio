Seattle Community Network (SCN)

Glow in the dark (GITD) NFC tags designed to promote non-profits. All models contain tags programmed to SeattleCommunityNetwork.org. Model meshes updated with public domain SVGs to display the Seattle Space Needle.

QR Code Tests

Initially I tried to print QR codes to promote SCN. However, getting a proper resolution with a 0.4mm nozzle is not practical considering the size of the tags.

QR Code Blend

QR codes were generated as SVGs with Python QR code library and 30% error. The center mesh was manually removed in blender to add logos.

Slicer Issues with QR Codes
